# aula-digital-bogota-moodle
Plataforma LMS basada en Moodle para el fortalecimiento educativo en contextos vulnerables
Aula Digital Bogotá - Plataforma LMS Moodle

Descripción del Proyecto:
Este proyecto tiene como objetivo la implementación de una plataforma LMS (Learning Management System) basada en Moodle, orientada al fortalecimiento de los procesos de enseñanza-aprendizaje en contextos educativos urbanos vulnerables de Bogotá.
La solución busca mejorar el acceso a recursos educativos, optimizar la gestión académica y fortalecer la interacción entre docentes y estudiantes mediante el uso de herramientas digitales.

Objetivo.
Implementar una plataforma LMS mediante la instalación, configuración y validación de Moodle, con el fin de apoyar los procesos educativos y promover el aprendizaje autónomo.

Metodología.
El desarrollo del proyecto se basa en:
 - Enfoque CDIO (Concebir, Diseñar, Implementar y Operar)
 - Metodología ágil Scrum para la gestión del desarrollo
 - Investigación aplicada con enfoque mixto

Tecnologías utilizadas:
 - Moodle (LMS)
 - PHP
 - MySQL / PostgreSQL
 - Servidor web (Apache)
 - HTML, CSS

Funcionalidades del sistema:
 - Registro e inicio de sesión de usuarios
 - Gestión de cursos virtuales
 - Carga de contenidos educativos
 - Actividades evaluativas (quices, tareas, foros)
 - Seguimiento del progreso académico
 - Generación de reportes

Estado del proyecto:
El proyecto se encuentra en fase de prototipo funcional con nivel de madurez tecnológica TRL5, validado en un entorno simulado mediante pruebas piloto con usuarios.
